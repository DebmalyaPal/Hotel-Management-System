# Hotel-Management-System

## Tech Stack :
#### Backend  : Java, Spring Boot, Spring Security, Spring Data JPA, JUnit, Mockito, Swagger
#### Frontend : ReactJs, BootStrap
#### Database : H2 (In-Memory Relational DB)
#### Tools    : Eclipse, VS Code, Postman
